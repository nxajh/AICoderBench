# 写时复制动态数组

## 题目描述

实现一个带引用计数的写时复制（Copy-on-Write, COW）动态数组。克隆操作为 O(1)——仅增加引用计数。对共享数组执行写操作时会触发深拷贝，确保其他引用不受影响。

## 接口

完整接口见 `solution.h`：

- `cow_create(capacity)` — 创建指定容量的数组，初始值全为零，引用计数为 1
- `cow_clone(arr)` — 浅拷贝（引用计数 +1），共享底层数据
- `cow_destroy(arr)` — 引用计数减一；仅当引用计数降为 0 时释放数据
- `cow_get(arr, index, out)` — 读取指定索引的值到 `*out`；成功返回 0，出错返回 -1
- `cow_set(arr, index, value)` — 写入值；如果引用计数 > 1，先执行深拷贝（触发 COW）；成功返回 0，出错返回 -1
- `cow_refcount(arr)` — 当前引用计数

## 要求

1. `cow_clone` 为 O(1)——不复制数据
2. 对共享数组（引用计数 > 1）执行 `cow_set` 时必须先深拷贝再修改
3. COW 后，写入方拥有引用计数为 1 的独立数据；其他引用保持原有数据
4. 对任意引用调用 `cow_destroy` 都是安全的，不受销毁顺序影响
5. 使用原子引用计数保证线程安全
6. 越界访问返回 -1
