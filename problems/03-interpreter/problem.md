# 递归下降表达式解释器

## 题目描述

实现一个递归下降解析器，用于解析和求值算术表达式。支持运算符优先级、内置函数和变量赋值。

## 接口

完整接口见 `solution.h`：

- `interp_create()` — 创建解释器实例
- `interp_destroy(interp)` — 释放资源
- `interp_eval(interp, expr, error_buf)` — 求值表达式字符串，返回结果（出错时返回 NaN，error_buf 填入错误信息）

## 支持的特性

1. **算术运算**：`+`、`-`、`*`、`/`、`%`、`**`（幂运算，右结合）
2. **比较运算**：`>`、`>=`、`<`、`<=`、`==`、`!=`
3. **逻辑运算**：`&&`、`||`、`!`
4. **括号**：`(...)`
5. **函数**：`max(a,b)`、`min(a,b)`、`abs(x)`、`round(x)`
6. **变量**：`x = 10`（赋值返回值），后续表达式可以引用 `x`
7. **一元运算**：`-expr`、`+expr`、`!expr`

## 错误处理

- 除以零 → NaN + 错误信息
- 未定义变量 → NaN + 错误信息
- 括号不匹配 → NaN + 错误信息
- 语法错误 → NaN + 错误信息
